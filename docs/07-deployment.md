## Colab Notebook (training/train_grpo.ipynb)

Cells in order:

1. **Title + Overview** (markdown) — what this notebook does
2. **Install** — `pip install unsloth trl bitsandbytes openenv-clarify-rl`
3. **Mount Drive (optional)** — for checkpoint persistence
4. **Imports** — model, trainer, env client
5. **Load model** — Qwen2.5-1.5B-Instruct + Unsloth 4-bit + LoRA
6. **Load env** — local import (faster than HTTP during training)
7. **Define rollout + reward function** — wraps env in GRPO-compatible reward fn
8. **Smoke test** — 5 GRPO steps, verify it runs
9. **Full training** — 600 steps with logging
10. **Save plots** — reward_curve.png, loss_curve.png to `/content/plots/`
11. **Evaluate** — held-out 100 scenarios, generate per_task_bars.png + per_component.png
12. **Push to HF** — LoRA adapter to `anurag203/clarify-rl-qwen2.5-1.5b-grpo`

Notebook MUST be runnable end-to-end with no manual edits except API keys.

Pin link: `https://colab.research.google.com/github/anurag203/clarify-rl/blob/main/training/train_grpo.ipynb`
